# 🏦 Advanced Financial Loan Analysis Dashboard | Power BI

## 📌 Project Overview

Financial institutions process thousands of loan applications every day, generating large volumes of customer, lending, repayment, and credit-risk data. Traditional reporting methods often make it difficult to identify risk patterns, monitor portfolio performance, and understand borrower behavior from a single view.

This project presents an **interactive Financial Loan Portfolio Analysis Dashboard** built in **Power BI** that transforms raw loan data into actionable business insights. The dashboard enables stakeholders to monitor lending performance, evaluate borrower profiles, identify high-risk loan segments, and support data-driven decision-making through interactive visualizations.

---

# 🎯 Business Objectives

- ✔ Monitor overall loan portfolio performance.
- ✔ Track lending, funding, and repayment trends.
- ✔ Identify high-risk borrowers and loan categories.
- ✔ Analyze borrower demographics and repayment behavior.
- ✔ Support credit risk assessment and strategic lending decisions.

# 💡 Key Insights

- 📌 **Grade B and Grade C loans** contributed the highest charged-off loan amounts, indicating elevated credit risk.
- 📌 A significant number of **Verified borrowers** still defaulted, suggesting that verification alone is not sufficient to assess creditworthiness.
- 📌 **Mortgage holders within Grade C** represented the highest-risk borrower segment.
- 📌 Borrowers with longer employment history generally demonstrated stronger repayment performance.
- 📌 Loan Grade, Verification Status, Employment Length, and Home Ownership were identified as key factors influencing loan performance and credit risk.

---

# 💼 Business Recommendations

- 💰 Strengthen underwriting policies for higher-risk loan grades.
- 📊 Improve credit assessment by combining verification status with additional borrower risk indicators.
- 📍 Closely monitor Grade B and Grade C loan portfolios to reduce default exposure.
- 🎯 Use borrower demographics and repayment trends to improve future lending strategies and portfolio performance.
